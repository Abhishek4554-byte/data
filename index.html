DevOps Lifecycle

Plan → Develop → Build → Test → Release → Deploy → Operate → Monitor (continuous loop)

Plan Goals, backlog, roadmap
Develop Code, review, branch
Build Compile, package
Test Quality, performance
Release Version, changelog
Deploy Staging → Prod
Operate Reliability, SRE
Monitor Metrics, alerts

Plan

Define goals, scope, and user stories. Align stakeholders and prioritize the backlog with success metrics.

Develop

Implement features on branches with code reviews and unit tests. Keep changes small and frequent.

Build

Compile and package the application. Run static analysis and produce artifacts for downstream stages.

Test

Validate functionality, performance, and security using automated and targeted manual testing.

Release

Version artifacts, update changelogs, and prepare rollout plans with clear rollback procedures.

Deploy

Promote builds to staging and production with CI/CD. Use blue‑green or canary strategies to minimize risk.

Operate

Run and maintain services, execute SRE practices, and manage capacity, cost, and reliability.

Monitor

Track SLIs/SLOs, logs, and traces. Feed insights back to planning for continuous improvement.
